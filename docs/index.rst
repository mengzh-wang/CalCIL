Welcome to CalCIL's documentation!
==================================

**CalCIL** is a computational imaging library that provides gradient descent-based image reconstruction tools.
It is designed to be modular and extensible, with a focus on ease of use and reproducibility. CalCIL is built on top of JAX and Flax.
